01 · Executive Summary

Why the Control Plane Must Change

Many plants still rely on isolated control systems, fragile cloud links, and spreadsheets. Hyperlinks Space offers one hardened runtime: MQTT for live data, OPC UA for machine meaning, and CRDTs for sites that go offline. It installs on hardware you already own and keeps running when the network fails.

⚡

Protocol-First, Capital-Light

We ship software and reference designs, not chip fabs. An ESP32 module or a legacy Siemens PLC can join the network in an afternoon. Move from pilot to production without a data-center budget line.

🛡

Compute Where Work Happens

Alerts, inference, and sync run on gateways and shop-floor PCs—not in a data center three countries away. That means lower delay, lower power use, and continuity through outages.

🌐

One Console for Every Asset

Factories, drone routes, and bonded warehouses report into the same state graph. Operators replace many vendor portals with one contract for throughput, custody, and exceptions.

02 · Architecture Pillars

Five Protocols. One Operating Contract.

Each pillar is a production requirement encoded in the core runtime—from North Sea platforms to lunar ISRU lines.

MQTT

MQTT Telemetry Core

ISO pub/sub with 2-byte headers and QoS 0–2. Engineered for bursty RF links, plant EMI, and satellite handoffs where TCP backoff wastes contact time.

  • BVLOS drone corridors
  • Remote pipeline and pump arrays
  • LEO ground-station relay chains
OPC UA

OPC UA Semantic Bridge

Certified client/server maps Siemens, ABB, and Schneider tags into our state graph—variables, alarms, and methods without per-site custom drivers.

  • Brownfield plant integration
  • OEM-agnostic operator HMIs
  • GxP and ISO audit trails
DTN

Delay-Tolerant Networking

NASA Bundle Protocol with store-and-forward buffers. Holds custody and telemetry through 4–22 minute signal delay and multi-day blackout windows.

  • Mars surface relay
  • Deep-space asset handoffs
  • Polar and maritime dead zones
CRDT

CRDT State Engine

Commutative merge lets rovers, vessels, and warehouses commit locally and reconcile later—no elected master, no split-brain after reconnect.

  • Autonomous haulage fleets
  • Multi-site inventory truth
  • Partition-tolerant ledgers
Edge AI

TinyModel Edge Partition

Splits vision and language workloads across ESP32, ARM gateways, and edge GPUs; bills inference by watt-hour at the site, not GPU-hour in a distant region.

  • Sub-100 ms anomaly detection
  • Federated model refresh
  • Neuromorphic reference firmware
